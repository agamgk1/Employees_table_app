 Homework INU
 
 In these app you can add new employees by clicking "add button", load .txt file with list of employees, 
 save new list to file and also generate a new report, sorted by work time.
 
 Technologies:

- Java 8/11
- JavaFX 8/11
- SceneBouilder
- css
